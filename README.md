# FORITUS - Future of Digital Innovation

A high-end futuristic IT company website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Cinematic Hero Section**: Full-screen dark futuristic background with particle effects.
- **Interactive Components**: Glassmorphism navbar, glowing service cards, and animated testimonials.
- **Tech Stack Marquee**: Infinite scrolling animation showcasing technologies.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile.

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js app router pages and layouts.
- `components/`: Reusable UI components (Hero, Navbar, Services, etc.).
- `public/`: Static assets like images and SVGs.
- `lib/`: Utility functions.
